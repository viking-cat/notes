# Setup

## IntelliJ Kotlin
* [Install Kotlin](https://kotlinlang.org/docs/getting-started.html#install-kotlin)

**NOTE!** Scroll down to bottom of download page and pick the IntelliJ IDEA Community Edition
  
## Kotlin CLI
* [Kotlin command-line compiler](https://kotlinlang.org/docs/command-line.html)
* [Setup VSCode for Kotlin Development](https://in-kotlin.com/ide/vscode/setup-vscode-for-kotlin-development/)
    * Go to [Kotlin Github](https://github.com/JetBrains/kotlin) page and download latest release of the compiler. It should be named something like *kotlin-compiler-2.2.21.zip*.
    * Unpack the zip file
